Общее описание.

	Для создания списка курсов используется система задач (Tasks). 
	При вызове создания списка курсов система возвращает идентификатор задачи,
	по которому можно отслеживать процесс создания курсов.

	Информация, которую можно получить через идентификатор задачи:
		* статус задачи
		* количество курсов, которые уже созданы
		* количество курсов, которые должны быть созданы по текущей задаче
		* список курсов (если задача была завершена успешно)

	Возможные статусы задач:
		 * NEW - список курсов получен, задача создана, но еще не начала выполняться
		 * IN_PROCESS - задача находится в процессе выполнения
		 * COMPLETED - задача была успешно завершена
		 * FAILED - не удалось завершить задачу из-за возникших ошибок
		 * WAITING_TO_CONTINUE - не удалось завершить задачу из-за ограничений GoogleClassroom по количеству курсов,
		   которые можно создать за один день. Выполнение данной задачи можно попытаться продолжить
		   при помощи api/Courses/retryTask/{taskId}. 
		 * CANCELLED - задача была отменена. В случае отмены задачи курсы, которые уже были созданы,
		   будут удалены и из локальной базы данных, и из GoogleClassroom. А создание тех курсов,
		   которые еще не были созданы, будет отменено.
		 * NOT_FOUND - не удалось найти задачу по заданному идентификатору


Детали реализации.

	Для реализации назначения и выполнения задач используется библиотека Quartz.net.

	Для создания списка курсов используются CoursesScheduler и CoursesCreator. 
	CoursesScheduler занимается управлением работой по созданию списка курсов. CoursesCreator непосредственно создает курсы из списка.

	Для управления отменой задач используются TaskCancellationScheduler и TaskCancellationExecutor по аналогии с предыдущим пунктом.


Предсозданные курсы.
	
	Для создания реальных курсов в процессе выполнения задачи используются предсозданные курсы. 
	Предсозданный курс кроме информации о курсе также содержит идентификатор задачи (к которой он относится),
	идентификатор реального курса (который создан по образу данного предсозданного курса или null, если курс еще не был создан).
	в процессе выполнения задачи из базы данных извлекаются предсозданные курсы и на их основе создаются реальные,
	после чего идентификатор реального курса записывается в предсозданный курс и этот предсозданный курс помечается как созданный (IsCreated=true).
